# Antigravity Instructions

Antigravity agents must use the same operating contract as other AI/dev agents.

Required reading:

- `AGENTS.md`
- `docs/AGENT_WORKFLOW.md`
- `docs/CURRENT_STATUS.md`
- `docs/MONEY_ROADMAP.md`

Do not perform high-risk actions without explicit owner approval:

- production deploys
- Firestore rules deployment
- payment, PayOS, bank, payout, or money changes
- env or secret changes
- mass publishing or automation enablement
